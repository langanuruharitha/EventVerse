## How It Works:

1. **User browses vendors** → Goes to any vendor detail page
2. **Clicks "🔖 Save Vendor"** → Saves to database
3. **Button changes to "✓ Saved"** → Visual feedback
4. **Goes to Dashboard** → Sees all saved vendors in one place
5. **Clicks vendor card** → Navigates back to vendor profile
6. **Clicks "✓ Saved" again** → Removes from saved list
